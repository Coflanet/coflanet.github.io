1. 계정 삭제

Coflanet 앱 내에서 직접 계정을 삭제할 수 있습니다.

삭제 방법

1앱 실행 후 하단 탭에서 My 행성 탭을 탭합니다.

2My 행성 화면 하단의 회원탈퇴를 탭합니다.

3확인 다이얼로그에서 탈퇴를 탭하면 계정이 즉시 삭제됩니다.

삭제되는 데이터

데이터 유형처리보관 기간
계정 정보 (소셜 로그인 식별자, 닉네임)즉시 삭제없음
커피 취향 설문 결과즉시 삭제없음
저장된 커피 원두 목록즉시 삭제없음
커피 추출 레시피즉시 삭제없음
앱 설정 및 환경설정즉시 삭제없음
계정 삭제 시 모든 데이터가 서버(Supabase)와 기기 로컬 저장소에서 영구적으로 삭제되며, 복구할 수 없습니다.

2. 데이터 삭제 (계정 유지)

계정을 삭제하지 않고도 앱 내에서 개별 데이터를 삭제할 수 있습니다.

삭제 가능한 데이터 및 방법

취향 설문 재설정: My 행성 > 설문 다시하기를 통해 기존 결과를 덮어씁니다.

원두 목록 삭제: 커피 원두 관리에서 개별 원두를 삭제할 수 있습니다.

전체 로컬 데이터 초기화: 앱 삭제 후 재설치 시 기기 내 모든 로컬 데이터가 삭제됩니다.

데이터 삭제 범위

데이터 유형개별 삭제방법
취향 설문 결과덮어쓰기My 행성 > 설문 다시하기
커피 원두 목록가능원두 관리에서 삭제
추출 레시피가능레시피 관리에서 삭제
계정 정보불가계정 삭제 필요
